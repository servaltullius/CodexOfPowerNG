## 6) Copy-Paste Description (EN)
```md
## Overview
Codex of Power NG is a new Skyrim SE/AE mod.
This is not a continuation of Codex of Power/SVCollection. It has been rebuilt around Prisma UI (Web UI) + SKSE DLL (C++).

## What You Can Do With This Mod
- Quickly find and register eligible items from your inventory
- Track registered items and progression in one place
- Monitor reward progression based on registration count and refund rewards when needed
- Adjust key gameplay/UI options directly in-game (protection, notifications, rewards, input, scale)
- Use large inventories more smoothly with pagination + virtualized lists

## Core In-Game Features
- **Quick Register tab**
  - Shows only safely consumable/registrable items
  - Supports search, paging, row selection, and Enter-to-register
- **Registered tab**
  - Displays all registered items
  - Supports search + refresh for fast verification
- **Rewards tab**
  - Shows cumulative reward stats/progression
  - Includes reward refund action
- **Settings tab**
  - Language, protection, reward, input scale, UI scale/performance options
  - Hotkey-driven open/close flow (default F4)
- **High-DPI support**
  - Auto/manual UI scaling
  - Input coordinate correction with `ui.inputScale`

## Compatibility
- Supported: Skyrim SE 1.5.97, Skyrim AE 1.6+
- Save compatibility: **Not compatible** with legacy Codex of Power/SVCollection
- Plugin form: **No ESP/ESM required** (SKSE DLL + Prisma UI view)

## Requirements
- SKSE64
- Address Library for SKSE Plugins
- Prisma UI
- Media Keys Fix SKSE

## Installation (MO2/Vortex)
1. Install `Codex of Power NG.zip` with your mod manager
2. Enable the mod
3. Launch the game and open the UI with the default hotkey (F4)

## Uninstall
- Disable/remove via mod manager
- Remove related files under `Data/SKSE/Plugins/CodexOfPowerNG*`

## Log Paths
- PrismaUI: `Documents/My Games/Skyrim Special Edition/SKSE/PrismaUI.log`
- CodexOfPowerNG: `Documents/My Games/Skyrim Special Edition/SKSE/CodexOfPowerNG.log`

## Permissions / Redistribution
Please ask for permission before redistributing or publishing modified versions of this mod's assets/code.
```
