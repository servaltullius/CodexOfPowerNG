# Codex of Power NG v1.0.1 (2026-02-08)

Patch release focused on Settings tab usability.

## Highlights
- Settings actions are now always visible without scrolling.
- Added an in-panel `Close` button next to `Reload` and `Save`.
- Added a regression test to keep sticky Settings actions stable.

## User-facing changes
- In the Settings tab, `Reload` / `Save` / `Close` stay reachable while scrolling long options.
- No gameplay logic changes; this update is UI/UX and stability focused.
