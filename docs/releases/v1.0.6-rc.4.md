# Codex of Power NG v1.0.6-rc.4 (2026-02-16, Pre-release)

## 요약
이번 프리릴리즈는 Prisma UI 뷰 스크립트를 모듈 단위로 분리해 유지보수성과 테스트 가능성을 높이는 데 집중했습니다.  
입력 보정 모듈 로드 실패 시를 대비한 fallback 경로도 추가했습니다.

## 주요 변경점
- `index.html`의 대형 inline 스크립트를 기능별 모듈로 분리
- 입력 보정 모듈(`input_correction.js`) 추가
- 입력 보정 모듈 미로드 시 fallback 입력 보정 경로 + 경고 로그 추가
- 모듈별 Node 테스트 신규 추가 및 기존 회귀 테스트 갱신

## 사용자 영향
- 기능 동작 변화 없음(주요 목적은 구조 개선/안정성 강화)
- 일부 환경에서 모듈 로드 실패 시에도 입력 보정이 완전히 비활성화되지 않음
