# Codex of Power NG v1.0.7-rc.1 (2026-02-18, Pre-release)

## 요약
이번 프리릴리즈는 저장 불러오기 이후 보상 기반 Actor Value(예: 소지 무게)가 일부 누락 반영될 수 있는 경로를 보완하는 안정화 업데이트입니다.  
로드 직후 보상 누락분만 재동기화하도록 보정해, 기존 플레이 진행과의 충돌 가능성을 낮췄습니다.

## 주요 변경점
- 로드 이후 `rewardTotals`와 실제 플레이어 AV 상태를 비교해 누락분만 보정 적용
- `kPostLoadGame`/`kNewGame` 메시지 경로에 보상 동기화 훅 추가
- 보상 동기화 델타 계산 규칙에 대한 C++ 호스트 테스트 추가
- 동기화 로직을 별도 헤더(`RewardsResync`)로 분리해 유지보수성 강화

## 사용자 영향
- 저장을 다시 불러왔을 때 보상 효과(특히 `CarryWeight`)가 누락되는 빈도가 줄어듭니다.
- 기존 등록/보상 시스템의 사용 방식은 변경되지 않습니다.
- 기능 동작 변화는 복구 안정성 영역에 한정됩니다.

## 기술 변경 상세
- 로드 후 보상 동기화 API 추가:
  - `include/CodexOfPowerNG/Rewards.h`
  - `src/Rewards.cpp`
- 동기화 계산 로직 모듈 추가:
  - `include/CodexOfPowerNG/RewardsResync.h`
- SKSE 로드 메시지 연동:
  - `src/main.cpp`
- 테스트 추가:
  - `tests/rewards_resync.test.cpp`
- 빌드 그래프 업데이트:
  - `CMakeLists.txt`

## 호환성/주의사항
- Codex of Power / SVCollection과 저장 호환되지 않습니다.
- 런타임 요구사항은 기존과 동일합니다:
  - SKSE64
  - Address Library for SKSE Plugins
  - Prisma UI
  - Media Keys Fix SKSE
- 본 프리릴리즈는 로드 후 보상 반영 안정성 보강이 목적이며, 신규 콘텐츠 추가는 없습니다.
