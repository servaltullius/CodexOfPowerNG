# Codex of Power NG v1.0.7-rc.10 (2026-02-18, Pre-release)

## 요약
이번 프리릴리즈는 `v1.0.7-rc.9`에 추가했던 소지무게 1회 복구 기능을 제거한 안정화 업데이트입니다.
임시 우회 기능을 제거하고 기존 보상/동기화 경로를 단순화했습니다.

## 주요 변경점
- Rewards 탭의 `Recover Carry +5` 버튼 제거
- `copng_recoverCarryWeight` 요청/핸들러 제거
- 1회 복구 상태 직렬화(`RFLG`, `carryWeightRecoveryUsed`) 제거
- 관련 테스트 및 직렬화 writer 흐름 원복

## 사용자 영향
- 소지무게 수동 복구 버튼은 더 이상 제공되지 않습니다.
- 기존 보상 지급/동기화 흐름은 유지됩니다.
- 저장 포맷은 복구 기능 추가 이전 경로로 정리됩니다.
