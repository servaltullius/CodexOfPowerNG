# Codex of Power NG v1.0.7-rc.11 (2026-02-19, Pre-release)

## 요약
이번 프리릴리즈는 보상 획득 주기의 기본값을 `10`에서 `5`로 조정한 튜닝 업데이트입니다.
새 설치/기본 설정 경로에서 더 자주 보상을 체감할 수 있도록 기본 정책을 맞췄습니다.

## 주요 변경점
- 기본 `rewardEvery` 값을 `10` -> `5`로 변경
- Settings 탭의 `rewardEvery` 폴백값을 `5`로 변경
- 기본 템플릿 `settings.json`의 `rewards.every` 값을 `5`로 변경
- Prisma UI JS API 문서 예시값을 `5`로 갱신

## 사용자 영향
- `settings.user.json`에 이미 `rewards.every`를 설정한 경우 기존 사용자 값이 그대로 유지됩니다.
- 새 설치/초기값 기준에서는 보상 롤이 기존 대비 더 자주 발생합니다.
