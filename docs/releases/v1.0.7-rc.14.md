# Codex of Power NG v1.0.7-rc.14 (2026-02-19, Pre-release)

## 요약
이번 프리릴리즈는 전투 중 비정상적으로 큰 데미지 수치가 발생할 수 있던 보상 동기화 경로를 안정화한 패치입니다.  
특히 `AttackDamageMult` 누적/동기화 구간에서 과도한 재적용이 발생하지 않도록 상한과 비수렴 가드를 추가했습니다.

## 주요 변경점
- `AttackDamageMult` 보상 총량 상한 추가 (`+100%`)
- 보상 동기화 패스에서 비수렴 AV 반복 적용 차단 가드 추가
- 관련 정책 테스트 갱신

## 사용자 영향
- 초반/중반 포함 전 구간에서 비정상적으로 큰 물리 데미지 폭증 가능성이 줄어듭니다.
- 일반적인 보상 획득 체감(등록→보상)은 유지되며, 극단적 누적만 제한됩니다.
- 기존 저장 방식/UI 사용 방식 변화는 없습니다.

## 기술 변경 상세
- 보상 상한:
  - `include/CodexOfPowerNG/RewardCaps.h`
- 보상 동기화 반복 적용 가드:
  - `src/Rewards.cpp`
- 정책 테스트:
