# Codex of Power NG v1.0.7-rc.16 (2026-02-19, Pre-release)

## 요약
이번 프리릴리즈는 저장 로드 직후 무기 공격력이 비정상적으로 크게 보였다가 무기 재장착 시 정상화되던 현상을 줄이기 위한 패치입니다.  
`AttackDamageMult` 동기화 후 장착 무기 능력치를 즉시 갱신하도록 보강해, 발도/납도 없이도 표시/계산이 안정되도록 조정했습니다.

## 주요 변경점
- 보상 full sync 종료 시 장착 무기 능력치(`UpdateWeaponAbility`) 강제 갱신 추가
- `AttackDamageMult` 보상 총량이 존재하면 로드 동기화 런에서 무기 능력치 갱신 플래그를 사전 활성화
- `AttackDamageMult` 보정이 실제 적용된 패스에서 무기 갱신 플래그를 유지/전파
- 관련 정책 테스트 보강

## 사용자 영향
- 저장 로드 직후 공격력이 과도하게 보였다가 무기 재장착 후 정상화되던 체감이 완화됩니다.
- 소지무게/저항/치명타 관련 기존 동기화 정책은 유지됩니다.
- 기존 UI/설정 사용 방식은 유지됩니다.
