# Codex of Power NG v1.0.7-rc.17 (2026-02-19, Pre-release)

## 요약
이번 프리릴리즈는 저장 로드 직후 무기 공격력이 비정상적으로 크게 보이는 문제를 줄이기 위해 무기 보상에서 `% 공격력(AttackDamageMult)` 채널을 제거하고, 기존 세이브 누적값을 1회 정리하는 마이그레이션을 추가한 패치입니다.

## 주요 변경점
- 무기 보상 테이블에서 `AttackDamageMult` 보상 제거
- 무기 보상은 고정값 스킬 증가(`OneHanded/TwoHanded/Archery`)로 대체
- 로드 동기화 첫 패스에서 기존 세이브의 `AttackDamageMult` 누적 총량을 상태에서 제거
- 관측 가능한 누적분이 있으면 actor 값에 역적용(`-removable`) 후 무기 능력치 갱신
- 정책 테스트 보강

## 사용자 영향
- 앞으로 획득하는 무기 보상에서 `% 공격력`은 지급되지 않습니다.
- 저장 로드 직후 공격력 수치가 비정상적으로 크게 유지되는 현상이 완화됩니다.
- 기존 세이브의 `% 공격력` 누적은 로드 동기화 시 1회 정리됩니다.
