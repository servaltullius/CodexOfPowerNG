# Codex of Power NG v1.0.7-rc.3 (2026-02-18, Pre-release)

## 요약
이번 프리릴리즈는 로드 이후 보상 Actor Value 동기화 경로에서 남아있던 과보정/스케줄 고착 리스크를 줄이기 위한 안정화 업데이트입니다.  
지속적으로 결손이 확인될 때만 보정하고, 동기화 작업이 멈춘 경우 watchdog으로 재시작하도록 보강했습니다.

## 주요 변경점
- AV 결손 보정에 연속 결손 조건(연속 3회) 추가로 일시 변동 구간 과보정 완화
- 동기화 스케줄러 stale 상태 감지 및 강제 재시작(watchdog) 추가
- 동기화 정책을 순수 함수 모듈로 분리해 테스트 가능성 강화
- 정책 회귀 테스트(`rewards_sync_policy`) 추가

## 사용자 영향
- 저장 로드 직후 소지무게/치명타 등 보상 AV가 늦게 반영되는 상황의 안정성이 개선됩니다.
- 일시적인 AV 변동(버프/로딩 중간값)에 의해 과도하게 보정되는 확률이 낮아집니다.
- 사용자 조작 방식/설정 구조 변화는 없습니다.
