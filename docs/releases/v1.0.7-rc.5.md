# Codex of Power NG v1.0.7-rc.5 (2026-02-18, Pre-release)

## 요약
이번 프리릴리즈는 보상 Actor Value 캡 적용 로직을 안정화하는 업데이트입니다.
이동속도 캡 단위를 수정하고, 로드 직후 캡 정규화가 초과분 하향-only로 동작하도록 바꿔 과보정 리스크를 낮췄습니다.

## 주요 변경점
- `SpeedMult` 보상 캡 단위를 `50.0`에서 `0.50`으로 수정
- 로드 시 캡 정규화 로직을 스냅샷 기반 하향-only 보정으로 변경
- 캡 정규화 계산 헬퍼(`ComputeCapNormalizationDeltaFromSnapshot`) 추가
- 보상 캡 정책 회귀 테스트 보강(JS) + 신규 C++ 테스트 추가

## 사용자 영향
- 치명타/저항/이동속도 보상값이 캡 정책에 맞게 더 일관되게 유지됩니다.
- 로드 직후 기존 과누적값이 있는 경우, 초과분만 즉시 내려 과보정 가능성을 줄입니다.
- 부족분 보정은 기존 동기화 정책(연속 결손 기준)을 유지합니다.
