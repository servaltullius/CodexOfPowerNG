# Codex of Power NG v1.0.7-rc.8 (2026-02-18, Pre-release)

## 요약
이번 프리릴리즈는 `v1.0.7-rc.7` 이후 코드리뷰 지적사항(quick resync 요청 유실, carry 동기화 판정 편향)을 반영한 안정화 업데이트입니다.
CarryWeight quick resync 재요청 플래그를 추가하고, carry delta를 current/snapshot 혼합 신호로 계산하도록 보강했습니다.

## 주요 변경점
- CarryWeight quick resync에 rerun 플래그 추가(요청 유실 방지)
- CarryWeight quick resync 상태 완료 시 rerun 요청이 있으면 즉시 재실행
- CarryWeight 동기화 delta를 current/snapshot 혼합(`ComputeCarryWeightSyncDelta`)으로 통합
- 동기화 헬퍼 함수(`MergeRewardSyncDeltaCandidates`, `ComputeCarryWeightSyncDelta`) 추가
- 회귀 테스트 보강(JS/C++)

## 사용자 영향
- 짧은 간격으로 연속 보상 발생 시 CarryWeight 재동기화 누락 확률이 줄어듭니다.
- CarryWeight 보상 반영 판정이 더 안정적으로 동작합니다.
- 기존 설정/저장 포맷 변경은 없습니다.
