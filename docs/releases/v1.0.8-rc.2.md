# Codex of Power NG v1.0.8-rc.2 (2026-02-21, Pre-release)

## 요약
이번 프리릴리즈는 보상 동기화 과정에서 외부 버프/장비 효과까지 “캡 초과”로 오인해 하향 보정(디버프처럼 보이는 감소)을 적용하던 문제를 수정한 안정화 업데이트입니다.

## 주요 변경점
- `ResistFrost`, `SpeedMult` 등 캡이 있는 ActorValue 동기화에서 “현재값이 캡을 넘는다”는 이유만으로 하향 보정(cap-guard)을 하지 않도록 변경
- 캡 초과(외부 효과) 상태에서 보상 동기화가 음수 델타를 만들지 않는 회귀 테스트 추가

## 사용자 영향
- 등록 수가 많아 보상 동기화가 자주 발생하는 환경에서도, 외부 버프/장비로 올린 냉기 저항/이동속도가 로드/동기화 후 갑자기 깎이는 현상이 완화됩니다.
- Rewards 탭에 표시되는 보상 누적/환불 동작은 기존과 동일합니다.
