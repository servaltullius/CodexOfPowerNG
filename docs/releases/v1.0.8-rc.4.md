# Codex of Power NG v1.0.8-rc.4 (2026-02-23, Pre-release)

## 요약
이번 프리릴리즈는 보상 동기화/Prisma UI 요청 처리 경로를 모듈 단위로 리팩터링해 유지보수성과 회귀 검증 신뢰도를 높인 안정화 업데이트입니다.

## 주요 변경점
- Reward sync 런타임 상태(세대, 예약 상태, rerun 플래그, watchdog 판단)를 `RewardsSyncRuntime`으로 분리
- Reward sync 엔진(캡 정규화, 레거시 마이그레이션, 패스 적용, 무기 능력 갱신)을 `RewardsSyncEngine`으로 분리
- Prisma UI register/undo 요청 처리의 JSON 파싱/결과 반영/패널 refresh 중복 로직을 공통 헬퍼로 통합
- Settings 저장 워커를 역할별 함수로 분해하고, 실패 롤백 및 UI 피드백 흐름을 명확화
- Reward sync runtime 전용 C++ 테스트(`rewards_sync_runtime`)를 추가

## 사용자 영향
- 의도된 게임플레이 동작 변경은 없습니다.
- 설정 저장 실패 시 롤백/피드백 경로가 명확해져 드문 타이밍 이슈 대응 안정성이 향상됩니다.
- 내부 구조 분리로 향후 보상/동기화 수정 시 회귀 위험이 낮아집니다.
