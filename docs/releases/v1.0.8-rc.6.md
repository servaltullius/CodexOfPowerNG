# Codex of Power NG v1.0.8-rc.6 (2026-02-23, Pre-release)

## 요약
이번 프리릴리즈는 스킬 숙련 보상 누적 시 부동소수 오차가 UI/퍽창 표기에 노출되던 문제를 완화한 안정화 패치입니다.

## 주요 변경점
- 스킬 ActorValue(One-Handed/Two-Handed/Archery 등)는 상태 누적은 유지하되, 실제 Actor 반영값은 정수 레벨 단위로 정규화
- 보상 기록(`rewardTotals`)과 실제 Actor 적용 델타를 분리해 소수점 잔여치가 직접 누적 적용되지 않도록 보정
- Undo 롤백과 Reward sync 스냅샷도 동일한 “실제 적용 총량” 기준으로 정합성 유지
- 숙련도 정규화 경로 회귀 테스트 추가

## 사용자 영향
- 스킬 숙련도 표기에서 `15.100000038147` 같은 비정상 소수점 노이즈가 줄어듭니다.
- 스킬 보상은 내부적으로 소수 누적을 유지하므로 장기 보상 진행도는 그대로 유지됩니다.
- 비스킬 보상(체력/저항/이속 등) 동작에는 의도된 변경이 없습니다.

## 기술 변경 상세
- 스킬 AV 판별/적용 총량 정규화 유틸 추가:
  - `include/CodexOfPowerNG/RewardCaps.h`
- 보상 기록 단계에서 상태 델타와 실제 Actor 적용 델타 분리:
  - `src/RewardsInternal.h`
  - `src/RewardsCore.cpp`
- Undo 시 롤백 Actor 델타를 정규화 총량 기준으로 계산:
  - `src/Rewards.cpp`
- Reward sync 스냅샷도 정규화 총량 기준으로 계산:
  - `src/RewardsSyncEngine.cpp`
- 회귀 테스트 보강:
  - `tests/reward_caps_policy.test.cjs`
  - `tests/undo_registration_flow.test.cjs`
