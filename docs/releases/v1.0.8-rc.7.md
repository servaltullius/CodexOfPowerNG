# Codex of Power NG v1.0.8-rc.7 (2026-02-23, Pre-release)

## 요약
이번 프리릴리즈는 아키텍처 심층 리뷰에서 식별된 고위험 안정성 이슈(UI 워커 재진입/스레드 경계/코세이브 로드 내성)를 우선적으로 보강한 안정화 패치입니다.

## 주요 변경점
- Prisma UI close/focus 워커를 generation 가드 기반으로 재구성해 재진입 시 블로킹 `join` 경로를 제거했습니다.
- 메인 스레드 작업 큐가 없을 때 등록/Undo/환불 로직을 동기 실행하지 않고 안전하게 요청을 중단하도록 변경했습니다.
- 코세이브 로드 시 reward/undo 레코드에 상한(cap)과 ActorValue 허용 목록 검증을 추가해 손상 데이터 내성을 강화했습니다.
- 위 변경에 대한 회귀 테스트 3개를 추가했습니다.

## 사용자 영향
- 로드 경계/빠른 토글 상황에서 UI 닫기 중 멈춤(프리즈) 가능성이 줄어듭니다.
- 드물게 스케줄러 인터페이스가 불능일 때, 기존처럼 위험한 동기 처리 대신 오류 토스트 후 재시도가 필요합니다.
- 손상/비정상 코세이브 데이터가 있더라도 전체 로드를 중단하기보다 위험 엔트리를 건너뛰도록 동작합니다.
