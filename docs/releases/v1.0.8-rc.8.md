# Codex of Power NG v1.0.8-rc.8 (2026-02-23, Pre-release)

## 요약
이번 프리릴리즈는 WSL 교차 빌드 환경에서 `ctest`가 오래된 테스트 항목을 참조해 실패하던 문제를 해결한 빌드/검증 안정화 패치입니다.

## 주요 변경점
- `CMakeLists.txt`에 `include(CTest)`를 추가해 테스트 메타파일 갱신 경로를 명확히 했습니다.
- 테스트 등록 조건을 `if(BUILD_TESTING AND COPNG_BUILD_CPP_TEST_TARGETS)`로 보강해, 테스트 타깃 OFF 시 stale test 등록이 남지 않도록 수정했습니다.

## 사용자 영향
- 런타임 동작/세이브 데이터/UI 기능에는 직접적인 변경이 없습니다.
- 개발/CI 관점에서 `wsl-release` 프리셋의 `ctest` 결과가 환경에 맞게 안정적으로 동작합니다.

## 기술 변경 상세
- CTest 연동 조건 보강:
  - `CMakeLists.txt`
