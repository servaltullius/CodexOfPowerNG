# Codex of Power NG v1.0.8 (2026-02-23)

## 요약
v1.0.8 정식 릴리즈는 1.0.8-rc 트랙에서 확인된 안정화 패치를 통합하고, 유저 피드백으로 보고된 **체력/매지카/스태미나 보상 누적(무한 증가처럼 보이는) 문제**를 우선 수정한 버전입니다.

## 주요 변경점
- **핵심 버그 수정**: 로드 시점에 `Health/Magicka/Stamina` 보상 동기화가 반복 적용될 수 있던 경로를 차단했습니다.
- **보상 표기/적용 정합성 개선**: 스킬 보상 적용값을 정수 레벨 기준으로 정규화해 `15.100000038147` 같은 부동소수 노이즈 노출을 줄였습니다.
- **UI 안정성 강화**: Prisma UI close/focus 워커를 generation 가드 기반으로 재구성해 로드 경계/빠른 토글 시 프리즈 가능성을 낮췄습니다.
- **요청 처리 안전성 강화**: 메인 스레드 큐가 불능일 때 등록/Undo/환불 로직을 동기 실행하지 않고 안전하게 요청을 중단하도록 변경했습니다.
- **세이브 로드 내성 강화**: 코세이브 reward/undo 레코드에 상한(cap)과 ActorValue 허용 목록 검증을 추가해 손상 데이터 대응을 강화했습니다.

## 사용자 영향
- 보고된 “게임 재시작 후 기존 보상 위에 새 보상이 계속 겹쳐 스탯이 비정상적으로 커지는” 증상의 재발 가능성이 크게 줄어듭니다.
- 이미 누적이 발생한 세이브는 **Rewards 탭 → Refund**를 1회 실행한 뒤 저장/재시작하면 정리에 도움이 됩니다.
  - Refund는 “기록된 보상 수치”만 되돌리며, 소비한 아이템/등록 기록은 복구하지 않습니다.
- 일반적인 UI/등록/Undo 흐름의 체감 동작은 기존과 동일하게 유지됩니다.
