# Codex of Power NG v1.0.9-rc.1 (2026-02-23, Pre-release)

## 요약
이번 프리릴리즈는 유저 제보로 확인된 **체력/매지카/스태미나 보상 재누적(로드 반복 시 스탯이 계속 증가)** 문제를 우선 차단하는 핫픽스입니다.

## 주요 변경점
- base-sticky AV(`Health`, `Magicka`, `Stamina`) 동기화에 전용 가드를 추가해, 로드 시 persistent 근거가 없으면 양수 재적용을 막도록 수정했습니다.
- 해당 AV의 보상 복구 계산에서 `current-base` 노이즈(외부/일시 버프)를 양수 누락 근거로 사용하지 않도록 분리했습니다.
- 회귀 테스트를 추가해 “current 값이 흔들려도 permanent 근거가 없으면 재적용 금지”를 검증했습니다.

## 사용자 영향
- 보고된 “게임 재시작 때마다 HP가 계속 커지는” 증상의 재발 가능성이 크게 줄어듭니다.
- 이미 수치가 누적된 세이브는 Rewards 탭의 **Refund** 1회 실행 후 저장/재시작을 권장합니다.
- 일반 등록/Undo/UI 흐름의 의도된 동작 변화는 없습니다.

## 기술 변경 상세
- base-sticky 동기화 정책/계산 함수 추가:
  - `include/CodexOfPowerNG/RewardsResync.h`
- reward sync 엔진의 base-sticky 경로를 전용 계산으로 전환:
  - `src/RewardsSyncEngine.cpp`
- 회귀 테스트 추가:
  - `tests/rewards_resync.test.cpp`
