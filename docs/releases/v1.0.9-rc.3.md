# Codex of Power NG v1.0.9-rc.3 (2026-02-24, Pre-release)

## 요약
이번 프리릴리즈는 보상 동기화/설정 저장/UI 갱신 경로의 복원력을 높이는 안정화 패치입니다.

## 주요 변경점
- reward sync에서 메인 task scheduler 불가 시 동기 fallback 루프를 제거하고 rerun 예약 중심으로 fail-closed 처리하도록 변경했습니다.
- 설정 저장 실패 시 현재 메모리 스냅샷이 아니라 마지막 디스크 반영 스냅샷으로 롤백하도록 수정했습니다.
- 메인 큐 드롭으로 UI 패널 갱신이 누락되면 pending refresh를 기록하고 다음 JS 요청에서 자동 재동기화하도록 보강했습니다.
- `ComputeCarryWeightSyncDelta`의 미사용 인자 경고를 정리하고, 관련 호스트 테스트 기대값을 현재 동작과 정합되게 갱신했습니다.

## 사용자 영향
- 로드/전환 경계에서 보상 동기화 경로가 더 안전하게 동작하며 프리징/히치 위험이 줄어듭니다.
- 설정 저장 실패 시 런타임 상태와 `settings.user.json` 불일치 가능성이 줄어듭니다.
- 일시적 큐 불가 상황 이후 UI 목록/보상/Undo 패널이 자동으로 회복됩니다.
