# Codex of Power NG v1.1.0-rc.3 (2026-03-06, Pre-release)

## 요약
이번 프리릴리즈는 빠른 등록에서 일부 퀘스트 물품이 목록에 보이거나 소비될 수 있다는 피드백을 반영한 안전성 보강 패치입니다.

## 주요 변경점
- 빠른 등록 목록 생성 단계에서 `InventoryEntryData::IsQuestObject()`를 확인해 엔진 기준 퀘스트 물품을 제외하도록 보강했습니다.
- 실제 등록 직전에도 동일한 퀘스트 물품 판정을 다시 확인해, 목록 누락/경합 상황에서도 소비를 막도록 했습니다.
- Prisma UI 도움말 문구를 갱신해 진행 중인 퀘스트 아이템이 등록 대상에서 제외된다는 점을 명시했습니다.
- 소스 검증 테스트를 추가해 퀘스트 물품 보호 체크가 목록/등록 두 경로 모두에 존재하는지 고정했습니다.

## 사용자 영향
- 진행 중인 퀘스트 아이템이 빠른 등록 목록에 덜 노출되고, 등록 시도 시에도 더 보수적으로 차단됩니다.
- 일반 등록/보상/Undo 흐름의 의도된 동작 변화는 없습니다.

- Codex of Power / SVCollection과 저장 호환되지 않습니다.
- 런타임 요구사항은 기존과 동일합니다:
