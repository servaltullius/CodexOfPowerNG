# Codex of Power NG v1.1.0-rc.4 (2026-03-08, Pre-release)

## 요약
이번 프리릴리즈는 PrismaUI 구조 분리와 런타임 상태 경계 정리를 마무리하고, 이를 release smoke/host 회귀 테스트까지 연결한 운영성 강화 빌드입니다.

## 주요 변경점
- PrismaUI를 `상태`, `입력`, `배선`, `렌더링`, `i18n` 모듈로 분리해 `index.html`을 조립 중심으로 단순화했습니다.
- `Reward`, `Notified`, `Serialization` 상태 저장소에 pure `Ops` 경계를 추가해 host-friendly C++ 테스트로 직접 검증할 수 있게 했습니다.
- `Registration`/`Rewards`/`Serialization`/`PrismaUI` 경계 리팩터링과 회귀 테스트를 여러 패스로 정리해 후속 병렬 작업 충돌을 줄였습니다.
- GitHub Actions에 release archive smoke 검증을 추가해, 새 UI 모듈과 필수 설정/lang 파일이 빠진 채 릴리즈되는 회귀를 잡도록 했습니다.

## 사용자 영향
- 의도된 게임플레이 규칙 변경보다는 유지보수성과 릴리즈 안정성이 크게 개선된 빌드입니다.
- PrismaUI 내부 구조가 바뀌었지만, 사용자 입장에서는 기존 Quick Register / Rewards / Undo / Settings 흐름을 그대로 사용하는 것이 목표입니다.
- 릴리즈 패키지에 모듈화된 UI 스크립트와 현재 설정 파일 구성이 일관되게 포함됩니다.
