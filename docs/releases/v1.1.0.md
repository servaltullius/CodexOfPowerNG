# Codex of Power NG v1.1.0 (2026-03-09)

## 요약
v1.1.0 정식 릴리즈는 1.1.0-rc 트랙에서 진행한 PrismaUI 구조 분리와 릴리즈 운영성 보강을 확정하고, 마지막으로 비활성 탭 렌더 억제와 가상 리스트 갱신 튜닝을 묶어 사용자 체감을 다듬은 버전입니다.

## 주요 변경점
- **정식 반영**: PrismaUI 모듈 분리, 상태 저장소 `Ops` 경계, release smoke 패키징 검증을 정식 릴리즈에 확정 반영했습니다.
- **UI 갱신 최적화**: Native 상태 갱신 시 현재 활성 탭만 다시 그리도록 바꿔, Quick/Register/Rewards/Undo 간 불필요한 교차 렌더를 줄였습니다.
- **가상 리스트 튜닝**: Quick/Register 목록의 overscan과 렌더 스로틀 값을 조정해 새로고침 직후 수렴감과 스크롤 체감을 다듬었습니다.
- **게임플레이 규칙 유지**: 이번 릴리즈는 유지보수성과 UI 응답성 개선이 중심이며, 의도된 보상/등록 규칙 변경은 없습니다.

## 사용자 영향
- 아이템 등록/등록 해제/보상 상태 갱신이 발생해도 보이지 않는 탭까지 모두 다시 그리던 비용이 줄어, UI 반응이 더 안정적으로 느껴질 가능성이 높습니다.
- Quick/Register 탭의 긴 목록에서 갱신 직후 빈칸 재정렬이나 지연 수렴이 덜 거슬리도록 조정했습니다.
- 기능 동작 자체는 기존 1.1.0-rc 트랙과 같은 방향이며, 저장 호환 정책도 달라지지 않습니다.
