# Codex of Power NG v1.2.0-rc.2 (2026-03-17, Pre-release)

## 요약
이번 프리릴리즈는 `v1.2.0-rc.1`에서 공개된 빌드 시스템을 더 깊게 확장하는 후속 빌드입니다. build catalog 확장, 점수 기반 슬롯 성장, 카탈로그 탐색 UI 재구성, 그리고 slot competition 밸런스 조정이 한 묶음으로 들어갑니다.

## 주요 변경점
- **빌드 카탈로그 확장**: 공격/방어/유틸 전반에 runtime이 실제로 처리하는 옵션 종류를 더 넓혔고, 저항/유틸 modifier/resource sustain 계열이 build 레이어 안으로 들어왔습니다.
- **점수 기반 슬롯 성장**: slotted option이 discipline build point에 따라 단계적으로 강해지도록 바뀌었고, Build 화면에서 현재 효과와 다음 단계 효과를 함께 읽을 수 있게 정리했습니다.
- **Build 탐색 UI 재구성**: discipline rail, theme tab, focused option detail rail 중심으로 Build 화면을 다시 짰고, 활성 슬롯 요약도 overflow-safe 구조 위에서 더 조밀하게 다듬었습니다.
- **slot competition 밸런스 재조정**: unlock pacing, weighted build point 비중, hybrid와 specialist 효율 차이를 다시 조정해 특정 테마 쏠림을 줄이는 쪽으로 정리했습니다.
- **활성 슬롯 밀도 개선**: 우측 하단 활성 슬롯 카드는 더 낮은 높이의 1행 카드로 압축되어, 선택 옵션 패널을 덜 가리지 않고 빠르게 훑는 요약 패널 역할에 더 가깝게 바뀌었습니다.

## 사용자 영향
- `rc.1`보다 해금 가능한 옵션 풀이 훨씬 넓어져, 실제 빌드 조립에서 선택지가 더 분명하게 느껴집니다.
- 같은 discipline 점수라도 어떤 옵션을 슬롯에 넣었는지에 따라 성장 체감이 더 직접적으로 갈립니다.
- Build 화면에서 `무슨 테마를 보고 있는지 / 지금 선택한 옵션이 어떤 상태인지 / 활성 슬롯이 어떻게 채워졌는지`를 읽는 흐름이 더 짧아졌습니다.
- 밸런스가 꽤 많이 움직인 프리릴리즈라서, `rc.1`과 비교하면 해금 템포와 강한 옵션 구도가 다르게 느껴질 수 있습니다.
